# photo-wall

图片展示墙

如遇样式丢失问题，请使用路径引入

```js
import PhotoWall from 'photo-wall-vt/dist/index'
```

## API

| 参数    | 说明   | 类型              | 默认 |
| ------- | ------ | ----------------- | ---- |
| options | 配置项 | ImgListItemType[] | []   |

#### ImgListItemType

| ImgListItemType | 说明     | 类型       | 默认 |
| --------------- | -------- | ---------- | ---- |
| title           | 标题     | string     | -    |
| name            | 图片名称 | string     | -    |
| img             | 图片url  | string\URL | -    |
